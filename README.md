# 🏆 Challenge Rules

- Solve at least **one LeetCode problem every day**
- Understand the problem before coding
- Write clean and readable Java code
- Optimize the solution whenever possible
- Push every solution to GitHub
- Share daily progress on LinkedIn

---

# 📖 Solution Format

Each solution includes:

- LeetCode Problem Number
- Problem Name
- Difficulty
- Topic
- Java Solution
- Time Complexity
- Space Complexity
- Approach (when required)

Example:

```text
Problems/
└── BitManipulation/
    └── PowerOfTwo.java
```
